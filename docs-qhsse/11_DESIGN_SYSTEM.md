# Design System

## Tokens
- Primary: `#2563eb` (biru, menggantikan emerald lama).
- Navbar: `#fdb913` (orange) hanya untuk header.
- Font: title 16px, group 15px, sub 13px.

## Stack Styling
- Tailwind CSS, theme di `tailwind.config.js`.
- Komponen React di `resources/js/Components`.

## Aksesibilitas
- Kontras memadai; label form jelas; status & due date menonjol (overdue merah).
