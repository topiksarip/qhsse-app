# Design System / UI Component Guide

## 1. Status Color

- Draft: gray
- Submitted: blue
- Under Review: indigo
- Approved: green
- Rejected: red
- In Progress: amber
- Waiting Verification: purple
- Closed: green
- Overdue: red
- Cancelled: gray

## 2. Component

- Button: primary, secondary, danger, ghost.
- Badge: status, severity, priority.
- Card: KPI, summary, alert.
- Table: sortable, filterable, paginated.
- Form: label, helper, error, required marker.
- Modal: confirmation, reject reason, upload.
- Toast: success, warning, error, info.

## 3. Accessibility

- Status tidak boleh hanya mengandalkan warna.
- Semua input punya label.
- Semua action penting punya confirmation.
- Keyboard navigation untuk form dan modal.
