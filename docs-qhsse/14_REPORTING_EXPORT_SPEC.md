# Reporting & Export Specification

## 1. Format

- CSV untuk data ringan.
- Excel untuk register dan matrix.
- PDF untuk laporan resmi.

## 2. Standard Export Filter

- Site
- Department
- Area
- Date range
- Status
- Category
- PIC/Owner

## 3. Report List

- Incident report PDF
- Investigation report PDF
- CAPA report Excel/PDF
- Inspection report PDF
- Audit report PDF
- Document register Excel
- Training matrix Excel
- Permit register Excel/PDF
- Environmental report Excel/PDF
- Security report Excel/PDF
- Quality NCR report
- Risk register
- Legal compliance register
- Emergency drill report
- Contractor performance report
- Asset certificate register
- Monthly QHSSE report

## 4. Monthly QHSSE Report Sections

- Executive summary
- Incident statistics
- CAPA status
- Inspection summary
- Audit finding
- Training compliance
- Document expiry
- Permit statistics
- Environmental compliance
- Security incidents
- Quality NCR
- Legal compliance
- Top overdue actions
