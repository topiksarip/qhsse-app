## 🚨 Report an Incident (All Employees)

**Quick Steps:**
1. Incidents → Create New
2. Fill: Date, Time, Location, Category, Severity
3. Describe what happened (be specific)
4. Add immediate action taken
5. Upload photos (if any)
6. Save Draft or Submit

**When to Report:**
- Accidents (injuries, damage)
- Near misses (close calls)
- Unsafe conditions (hazards)
- Unsafe acts (violations)
- Environmental spills

**Tips:**
- Report within 24 hours
- Be specific (who, what, when, where, how)
- Include witness names
- Take photos before cleanup (if safe)

---

## 👀 Review Incidents (QHSSE Officers)

**Quick Steps:**
1. Incidents → Pending Review
2. Click incident number
3. Review description + evidence
4. **Approve** (moves to investigation)
   - OR **Reject** (requires reason → notifies reporter)
5. Add comments if needed

---

## 🚪 Check-In Visitor (Security Officers)

**Quick Steps:**
1. Security → Visitors → Check In
2. Fill visitor information:
   - Name, Company, Purpose
   - Host Employee
   - Identity Type & Number (KTP/SIM/Passport)
   - Phone, Vehicle Plate
3. Click Check In
4. Print badge (if applicable)

**Check-Out:**
1. Security → Visitors → Checked In
2. Find visitor
3. Click Check Out
4. Collect badge

**Tips:**
- Verify identity document
- Notify host employee
- Issue visitor badge
- Brief on safety rules

---

## 📞 Customer Complaints (QHSSE Officers)

**Create:**
1. Quality → Complaints → Create New
2. Fill: Customer Name, Contact, Date
3. Describe issue (detailed)
4. Set Priority (Low/Medium/High/Urgent)
5. Add immediate action
6. Assign responsible person
7. Submit

**Close:**
1. Open complaint
2. Click Close Complaint
3. Enter Resolution (permanent fix)
4. Upload evidence (optional)
5. Click Close

---

## 📤 Upload Evidence

**Supported:**
- Images: JPG, PNG, GIF (max 10MB)
- Docs: PDF, DOC, DOCX, XLS, XLSX (max 10MB)
- Video: MP4 (max 50MB)

**Steps:**
1. Click Upload / Add Evidence
2. Choose file
3. Add description
4. Click Upload
5. Wait for confirmation
