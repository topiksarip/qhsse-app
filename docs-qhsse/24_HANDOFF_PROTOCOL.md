# Handoff Protocol

Setiap phase selesai wajib:
- Migration/model/request/controller/route/UI lengkap bila relevan.
- Permission backend diterapkan.
- File evidence via private file service.
- Nomor via numbering service (bila relevan).
- Status/workflow via workflow core (bila relevan).
- Audit trail & comment activity (bila relevan).
- Test happy/permission/edge + `npm run build` + `make test` hijau.
- Docs/handoff diperbarui (note: handoff/ lama dihapus per optimasi; ganti dengan `docs-qhsse`).
