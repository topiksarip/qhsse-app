# Test Plan & UAT

## 1. Test Level

- Unit/self-check untuk business logic penting.
- Integration test untuk workflow dan permission.
- UAT scenario untuk user.
- Regression smoke test per release.

## 2. Standard Test Per Modul

- Create record.
- Validation required field.
- Submit workflow.
- Approve/reject workflow.
- Permission by role.
- Data scoping by site/department/company.
- File upload/download.
- Comment/activity log.
- Notification trigger.
- Export.
- Audit trail.

## 3. UAT Sign-off

Setiap modul butuh:

- UAT date.
- Tester name/role.
- Scenario result.
- Defect list.
- Sign-off decision.

## 4. Core Acceptance Criteria

- Login/logout bekerja.
- Role permission berjalan.
- Master data bisa dikelola.
- File upload aman.
- Numbering unik.
- Audit trail tercatat.
- Notification dasar terkirim.
